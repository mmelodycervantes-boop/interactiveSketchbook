Micro Project Idea

Terms and Conditions

Core Rule and System Clarity - 20 pts

The core rule is that the user is not able to accept the terms and conditions unless they are reading at an acceptable pace. If the user scrolls fast to reach the end of the text, a warning would be given, but if they continue to scroll fast, then there is a penalty of 5 seconds, where the user is unable to scroll. The user must maintain a slow to medium scrolling pace (which is how the reading is detected) and reach the end of the text for the “Accept” button to turn green and clickable.

State Progression - 15 pts

The system moves through distinct structural states, each based on the user's interaction with the system, and each state changes what the user can do. For example, in the code, there are five clear states. 1. The Initial locked state, where the accept button is gray, prompting the user to read/scroll through the terms and conditions. 2. If the user is scrolling at a slow or medium speed (let slowThreshold = 25; let mediumThreshold = 60;) the system reads it as normal or that the user is reading the text. 3. When the user scrolls fast ( let fastThreshold = 120;), the system shows a pop-up warning, telling the user to read the terms and conditions. 4. If the user scrolls fast once again, then they face a 5-second penalty where the scrolling is frozen. 5. When the user reaches the end of the text, the system reaches an unlocked state, which causes the “Accept” button to turn green. Interaction and Evolution - 15pts

The user’s interactions does produce meaningful consequences and evolves over time. The system reponds to what the user does. The users input changes whether scrolling continues: fast scrolling causes a warning and faster scrolling causes the user to be in a 5 second timeout. The time-out and the forced paused scrolling creates restriction as the user is restricted from scrolling. The restriction helps to prompt the user read the terms and agreement. Once the user reaches the bottom of the text, the “Accept” button become unlocked and is able to be accepted.
