Sketch 9 - Photos of Blueberry

Images of Blueberry

Modularizing code made me think about how the user will interact with the code and if they would understand how to interact with it with a few words. A new interaction behavior that I added was the auto-advance mode and the dot indicator to signal which photo they were on.

The sketch communicates more clearly than Sketch #8, as it shows the user that 5 photos are using the dots as an indicator, and the timer bar allows the user to see that there is an auto advancement feature.
